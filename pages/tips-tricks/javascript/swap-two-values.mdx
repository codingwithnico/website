# Swap two values in Javascript

```jsx
let dog = 'Max'
let cat = 'Nini'

let temp = cat
cat = dog
dog = temp[
  // Using array destructuring
  (dog, cat)
] = [cat, dog]

// Result -> dog = "Nini", cat = "Max"
```
