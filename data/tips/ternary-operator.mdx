# Ternary operator control flow

```jsx
const isDev = true

// Normal if/else control flow
if (isDev) {
  console.log('I am a dev')
} else {
  console.log('Not a dev')
}

// Using ternary operator
isDev ? console.log('I am a dev') : console.log('Not a dev')
```
